Basic HTML Tags and Their Attributes

<html> - HTML Tag

Description:

  • The <html> tag represents the root of an HTML document.
  • In HTML tag we write all content but in this we have two section. which was define below.
    • <head>
    • <body>

Attributes:

  • lang - Specifies the language of the content.

Example:

<html lang="en">...</html>

<head> - head Tag

Description:

  • The HTML head is the contents of the <head> element.
  • Unlike the contents of the <body> element (which are displayed on the page when loaded in a browser), the head's content is not displayed on the page.
  • Instead, the head's job is to contain metadata about the document.

Attributes(there was no comman attribute):

Example:

<head>...</head>

Can Contain:

  • <title>
  • <base>
  • <link>
  • <meta>
  • <style>
  • <script>
  • <noscript>
  • <taemplate>

<meta> - meta Tag

Description:

  • The <head> tag provides metadata (data about data) about the HTML document.
  • It's placed inside the <head> tag and is not displayed on the webpage itself.

Attributes:

  • name - Specifies the type of meta element it is; what type of information it contain.
  • decription - Specifies the actual meta contain.
  • chatset - Define the character encoding
  • http-equiv - Provides an HTTP header

Comman Used of <meta>

  1. Character Encoding
    2. <meta charset="UTF-8">
  2. Page Description (SEO)
    3. <meta name="description" content="Best tutorial for learning HTML.">
  3. Keyword for SEO
    4. <meta name="keywords" content="HTML, CSS, Web Development">
  4. Author Info
    5. <meta name="author" content="dr.Computer">
  5. Viewport Settings
    6. <meta name="viewport" content="width=device-width, initial-scale=1.0">
  6. Referesh/Redirect
    7. <meta http-equiv="referance" content="dr.Computer">

<link> - link Tag

Definition:

  • The <link> tag define a relationship between the current document and an external resource.
  • It's most commonly used to link external stylesheets.
  • It is always placed inside the <head> tag.

Attributes:

  • rel - Specifies the relationship between the current document and the linked resouces. like stylesheet, manifest, icon
  • href - URL of the linked resouces. like .css, .icon
  • type - MIME type of the linked document (optional)
  • media - Specifies the media/device for which the styles are optimized

Example:

<link rel="stylesheet" href="style.css">

<script> - script Tag

Definition:

  • The <script> tag is used to embed or reference executable JavaScript code.
  • It can be placed in the <head> or near the end of the <body>.

Attributes:

  • src - Specifies the relationship between the current document and the linked resouces. like stylesheet, manifest, icon
  • type - URL of the linked resouces. like .css, .icon
  • defer - Defers execution until the HTML is fully parsed.
  • async - Executes the script asynchronously (does not block HTML parsing).

Example:

<link rel="stylesheet" href="style.css">
